Le Web

Le Web est un système d'information mondial reposant sur Internet. Le W3C est en charge de définir les standarts permettant sa mise en oeuvre.

Objectif

Ce site est réalisé dans le cadre de l'examen de l'UE MIA0301V portant sur la partie "Initiation à l'écriture de pages Web statiques"
